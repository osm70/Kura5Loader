# Kura5Loader

This tool lets you fake your time and location for the Bokti fangame Kura 5: Bonds of the Undying.

# Setup

Download the release zip, the Kura5 game and RunAsDate.

https://chickenhat.itch.io/kura5-bonds-of-the-undying

https://www.nirsoft.net/utils/run_as_date.html

Copy the Kura5 game files into the Data folder and copy RunAsDate into the same folder. Copy the entire Files folder into Kura5_Data inside the Data folder.

# Usage
Run Kura5 Loader.exe and follow select your virtual location (or pick 0 to use the real one.)
